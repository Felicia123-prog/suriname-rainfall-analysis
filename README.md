# suriname-rainfall-analysis
A toolkit for analyzing rainfall data from Suriname. This project loads monthly rainfall totals from multiple weather stations, generates yearly and multi‑year rainfall graphs, and produces statistics that answer key research questions about Suriname’s climate, seasons, and rainfall trends.
